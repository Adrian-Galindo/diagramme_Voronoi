1. Télécharger le projet puis ouvrir un terminal dans le dossier.

2. (Option recommandé) Lancer un environnement virtuel

   Windows :
   venv\Scripts\activate

   Linux / Mac :
   source venv/bin/activate

3. Installer les dépendances

   pip install -r requirement.txt

4. Lancer l’application

   python ./app/main.py
